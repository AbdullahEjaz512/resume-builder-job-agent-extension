# All-in-One Job Agent

A Chrome and Edge browser extension for building resumes, auto-applying to jobs on LinkedIn, and analyzing resume quality.

## Features
- **Resume Builder Wizard:** Step-by-step resume creation
- **Auto-Apply Agent:** Autonomously search and apply to jobs on LinkedIn
- **Resume Score Checker:** AI-powered resume analysis against job descriptions
- **Resume Upload:** Store and use your own resume file

## Installation (Local Development)
1. Clone: `git clone https://github.com/AbdullahEjaz512/resume-builder-job-agent-extension.git`
2. Install: `npm install`
3. Build: `npm run build`
4. Load in Chrome: `chrome://extensions` → Developer mode → Load unpacked → Select `dist/` folder

## Requirements
- Node.js 16+
- Ollama (for AI features)
